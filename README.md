# E-Commerce Database Schema Design

## Overview

This project is part of my SQL Developer Internship - **Task 1: Database Setup and Schema Design**.

The objective is to design a relational database for an **E-Commerce Management System** using Microsoft SQL Server.

## Features

* Created `ECommerceDB` database
* Designed normalized tables
* Implemented Primary Keys and Foreign Keys
* Applied constraints (`NOT NULL`, `UNIQUE`, `CHECK`, `DEFAULT`)
* Established relationships between tables
* Prepared an Entity Relationship (ER) Diagram

## Tables

* Customers
* Categories
* Suppliers
* Products
* Orders
* OrderDetails
* Payments

## Technologies Used

* Microsoft SQL Server
* T-SQL (SQL Server)

## Outcome

A well-structured relational database schema that demonstrates database design, table creation, relationships, and integrity constraints.
